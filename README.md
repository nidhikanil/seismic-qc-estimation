# seismic-qc-estimation

Python workflow for seismic Qc estimation using ambient noise cross-correlation, signal processing, and coda-wave analysis methods.

## Features
- Bandpass filtering and waveform preprocessing
- Hilbert envelope computation
- Surface-wave coda analysis
- Grid-search based Qc estimation
- Basic seismic visualization workflows

## Tools Used
- Python
- NumPy
- SciPy
- Matplotlib
- ObsPy

## Note
This repository contains a generalized educational/research workflow and does not include unpublished datasets or manuscript figures.
